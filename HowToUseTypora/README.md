## 💡 为什么选择 Markdown？

### Markdown 的优势

- ✍️ **简单易学**：纯文本格式，语法简洁，10 分钟上手
- 📝 **专注内容**：无需关心排版细节，专注于内容创作
- 🔄 **可转换**：可导出为 HTML、PDF、Word 等多种格式
- 🌐 **广泛应用**：GitHub、博客、文档、笔记都支持
- 💻 **跨平台**：任何设备、任何编辑器都可以编辑
- 🎯 **版本控制友好**：纯文本，Git 可以完美追踪变更

### 为什么选择 Typora？

- 👁️ **所见即所得**：实时预览，边写边看效果
- 🎨 **界面简洁**：专注写作，无干扰界面
- 📊 **支持图表**：内置数学公式、流程图支持
- 🖼️ **图片管理**：方便的图片插入和管理
- 🎯 **多种主题**：可自定义编辑器外观
- ⚡ **快捷键丰富**：提高编写效率

## 📖 Markdown 基础语法速览

**标题：**
```markdown
# 一级标题
## 二级标题
### 三级标题
```

**文本样式：**
```markdown
**粗体** *斜体* ~~删除线~~ ==高亮==
```

**列表：**
```markdown
- 无序列表
1. 有序列表
- [ ] 任务列表
```

**链接和图片：**
```markdown
[链接文本](https://example.com)
![图片描述](image.png)
```

**代码：**
````markdown
`行内代码`

```python
# 代码块
print("Hello, World!")
```
````

**表格：**
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容1 | 内容2 | 内容3 |
```

## ❓ 常见问题

### Markdown 和 Word 有什么区别？
- **Markdown**：纯文本，语法简单，专注内容，版本控制友好
- **Word**：所见即所得，功能强大，但文件格式复杂，不易版本控制

### Typora 是免费的吗？
Typora 需要购买许可证（约 $14.99），但提供 15 天免费试用。也可以使用免费替代品如 Obsidian、Mark Text 等。

### Typora 支持哪些平台？
Typora 支持 Windows、macOS 和 Linux。

### 图片如何管理？
建议在项目目录创建 `images/` 文件夹，使用相对路径引用图片。详见：[03 - Markdown 高级功能](./03-Markdown高级功能.md)

### 如何导出 PDF？
需要安装 Pandoc，然后在 Typora 中选择 `文件` -> `导出` -> `PDF`。详见：[04 - 实用技巧与最佳实践](./04-实用技巧与最佳实践.md)

### Markdown 能做数学公式吗？
可以！Typora 支持 LaTeX 数学公式。详见：[03 - Markdown 高级功能](./03-Markdown高级功能.md)
