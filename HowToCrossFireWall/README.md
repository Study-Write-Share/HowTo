## 什么是科学上网

科学上网是指通过代理服务器访问被限制的国际网站和服务，如 Google、GitHub、YouTube 等。

## 准备工作

在开始之前，你需要：

1. 一台能够联网的计算机（Windows/Mac/Linux）
2. 一个可靠的机场服务提供商账号
3. 对应的代理客户端软件

## 选择代理软件

### Windows 系统

#### 1. Clash for Windows（推荐）

**下载方式：**
- 访问 GitHub 发布页：https://github.com/clash-verge-rev/clash-verge-rev/releases
- 下载最新版本的 `.exe` 安装包
- 双击安装包完成安装

**使用步骤：**（以windows版本为例）

1. **安装完成后启动软件**
2. **导入订阅链接：**
   - 点击左侧 `订阅` 
   - 点击右上角 `新建` 
   - 在`订阅链接` 中输入机场提供的链接
   - 点击 `保存` 
3. **选择节点：**
   - 点击左侧 `Proxies` (代理)
   - 选择合适的节点（建议选择延迟较低的）
4. **开启代理：**
   - 点击左侧 `Settings` (设置)
   - 打开 `System Proxy` (系统代理) 开关

#### 2. V2rayN（推荐在手机上使用，但三星OneUI系统不推荐使用，亲测容易崩溃）

**下载方式：**

- GitHub 地址：https://github.com/2dust/v2rayN/releases
- 下载 `v2rayN-windows.zip` 
- 解压后运行 `v2rayN.exe`

**使用步骤：**

1. 右键任务栏图标
2. 选择 `订阅` -> `订阅设置`
3. 添加订阅地址
4. 更新订阅
5. 选择节点并启用系统代理

## 机场订阅指南

### 什么是机场

"机场" 是代理服务提供商的俗称，提供多个服务器节点供用户使用。

### 如何选择机场

选择机场时需要考虑：

1. **稳定性**：服务器是否经常掉线
2. **速度**：带宽是否充足，延迟是否低
3. **价格**：根据自己的预算选择合适的套餐
4. **节点数量**：是否提供多个地区的节点
5. **口碑**：查看其他用户的评价

### 订阅步骤

1. **注册机场账号**
   - 访问机场官网
   - 使用邮箱注册账号
   - 选择合适的套餐并购买

2. **获取订阅链接**
   - 登录机场用户中心
   - 找到 "订阅信息" 或 "订阅链接" 页面
   - 复制订阅链接（通常以 `https://` 开头的长链接）

3. **导入到客户端**
   - 打开代理软件
   - 找到订阅/配置管理功能
   - 粘贴订阅链接并更新

4. **选择节点开始使用**
   - 节点命名通常包含地区信息（如：香港、日本、美国等）
   - 选择延迟较低的节点
   - 开启系统代理

### 订阅链接安全提示

⚠️ **重要提示：**
- 订阅链接是你的专属链接，不要分享给他人
- 定期更新订阅以获取最新节点
- 如果链接泄露，及时在机场后台重置

## 常见问题

### 1. 连接后无法上网

**解决方法：**
- 检查系统代理是否已开启
- 尝试切换其他节点
- 检查订阅是否过期
- 重启代理软件

### 2. 速度很慢

**解决方法：**
- 选择延迟更低的节点
- 尝试不同地区的节点
- 检查本地网络是否正常
- 考虑升级机场套餐

### 3. YouTube/Netflix 无法播放

**解决方法：**
- 选择专门标注了流媒体解锁的节点
- 切换到香港、台湾或美国节点
- 清除浏览器缓存后重试

### 4. 订阅更新失败

**解决方法：**
- 检查订阅链接是否正确
- 尝试手动复制粘贴订阅链接
- 检查机场服务是否正常
- 联系机场客服

### 5. 代理软件无法启动

**解决方法：**
- 以管理员身份运行
- 关闭杀毒软件后重试
- 重新下载安装
- 检查是否有端口冲突

## 使用建议

1. **不要同时开启多个代理软件**，会造成冲突
2. **选择就近节点**，通常延迟更低
3. **定期更新订阅**，保持节点列表最新
4. **遵守当地法律法规**，仅用于学习和工作
5. **不要访问非法网站**，保护个人信息安全

## 进阶设置

### 分流规则

分流规则可以让：
- 国内网站直连（不走代理，速度更快）
- 国外网站走代理

大多数现代代理软件都内置了分流规则，通常选择 `规则模式` 即可自动分流。

### 代理模式说明

- **全局模式**：所有流量都走代理
- **规则模式**：根据规则自动分流（推荐）
- **直连模式**：不使用代理

## 注意事项

⚠️ **安全提醒：**
- 使用代理后，不要进行敏感操作（如网银转账）
- 选择可信赖的机场服务商
- 定期修改机场账号密码
- 不要在公共电脑上保存订阅信息
